Description:
This project is a banking system built with Python, Tkinter, and MySQL. 
It allows deposits, withdrawals, and transfers.

Instructions to Run:
1. Install Python and MySQL.
2. Restore the database by running:
   mysql -u root -p mybankbd < mybankbd.sql
3. Install dependencies:
   pip install mysql-connector-python tk
4. Run the application:
   python Main.py
